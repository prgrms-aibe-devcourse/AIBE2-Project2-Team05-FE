# 📝 TravelMate 프론트엔드 기획서 (PRD 기반 통합 버전)

---

## 1. 🎯 프로젝트 개요

**TravelMate**는 여행자들이 자신의 여행 계획을 공유하고, 여행 동행자를 매칭하며, 여행 피드에 대한 소통과 리뷰를 통해 여행 경험을 나누는 **소셜 여행 플랫폼**입니다.
프론트엔드 파트는 사용자 중심 UI 설계, 지도 기반 여행 등록, 피드 기반 인터랙션, 매칭/채팅/관리자 기능을 구현합니다.

---

## 2. 🧱 핵심 기술 스택

* **React**
* **TypeScript**

---

## 3. 🗂 전체 페이지 구성

| 페이지명              | 경로                           | 설명                       |
| ----------------- | ---------------------------- | ------------------------ |
| 홈                 | `/`                          | 메인 피드, 추천 사용자            |
| 회원가입              | `/signup`                    | 계정 생성                    |
| 로그인               | `/login`                     | 일반 사용자 로그인               |
| **관리자 로그인**       | `/admin/login`               | 관리자 전용 로그인               |
| 마이페이지             | `/mypage`                    | 내가 작성한 피드/리뷰/여행계획/매칭 보기  |
| **프로필 페이지**       | `/user/:userId`              | 사용자의 소개, 피드 리스트          |
| **프로필 피드 상세**     | `/user/:userId/feed/:feedId` | 피드 상세 보기                 |
| **여행계획 작성**       | `/plan/create`               | KakaoMap 기반, 장소/일정 입력    |
| **여행계획 결과**       | `/plan/:planId/view`         | 타임라인 형식의 여행 요약 결과 페이지    |
| **여행 피드 목록**      | `/feed`                      | 인스타그램 형식 전체 피드 (타 유저 포함) |
| 피드 상세             | `/feed/:feedId`              | 본문 + 댓글 + 좋아요 + 리뷰       |
| **피드 좋아요한 유저 목록** | `/feed/:feedId/likes`        | 게시물 좋아요 누른 사람 리스트        |
| **리뷰 작성**         | `/feed/:feedId/review/write` | 해당 피드에 대한 리뷰 작성          |
| **리뷰 목록**         | `/feed/:feedId/reviews`      | 피드에 대한 전체 리뷰 리스트         |
| **매칭 추천**         | `/match/recommend`           | 조건 기반 AI 매칭 추천           |
| 매칭 요청             | `/match/request/:userId`     | 동행 요청                    |
| **매칭 알림함**        | `/match/alerts`              | 받은 요청 확인 및 수락/거절         |
| 채팅 목록             | `/chat`                      | 참여 중 채팅방 리스트             |
| 채팅 상세             | `/chat/:roomId`              | WebSocket 기반 실시간 채팅      |
| 신고하기              | `/report/:type/:id`          | 유저, 피드, 댓글 등 신고          |
| 관리자 대시보드          | `/admin`                     | 관리자 콘텐츠/유저/신고 관리         |
| 404               | `*`                          | 잘못된 경로 예외 처리             |

## 4. 🔍 주요 기능별 구조

### 📍 여행계획 작성 및 결과

* **지도 기반 입력 (KakaoMap API 사용)**

  * 마커 찍기, 장소명/시간/설명 입력
  * 카테고리(식사/관광/숙박 등) 선택 가능
* **결과 페이지**

  * `/plan/:planId/view`
  * 일자별 타임라인 구성
  * 일정/예산/인원/카테고리/이미지 포함
  * 공유하기, 좋아요 기능, 리뷰 보기 포함

---

### 🖼 피드

* 여행계획 작성 시 자동 생성됨
* 피드 리스트는 인스타그램 구조 (`/feed`)
* 피드 상세에는:

  * 본문/이미지
  * 좋아요 (하트 UI)
  * 댓글
  * **리뷰 탭**
  * 좋아요 누른 사람 리스트 보기 (`/feed/:id/likes`)

---

### 📝 리뷰 (피드 기준)

* **리뷰는 사용자에 대한 것이 아니라, 피드에 대한 평가**
* 작성 경로: `/feed/:feedId/review/write`
* 조회 경로: `/feed/:feedId/reviews`
* 구성:

  * 별점, 내용, (선택) 사진
  * 리뷰 작성자는 매칭된 사용자 또는 여행에 참여한 사람
* 마이페이지에서는 `내가 작성한 리뷰` 확인 가능

---

### 🤝 매칭

* **추천**: `/match/recommend`

  * 위치/기간/스타일 기반 매칭 알고리즘 결과 표시
* **요청**: `/match/request/:userId`
* **알림함**: `/match/alerts`

  * 받은 요청 리스트, 수락/거절
* **매칭 확정 시 채팅방 자동 생성**

---

### 💬 채팅

* 채팅방 리스트: `/chat`
* 실시간 채팅: `/chat/:roomId` (WebSocket 기반)
* 채팅방 자동 생성 트리거: 매칭 확정 시

---

### 🛡 관리자 기능

* 로그인: `/admin/login`
* 대시보드: `/admin`

  * 유저 관리: 차단/삭제/노트
  * 피드/댓글 블라인드
  * 신고 처리 및 통계
  * 리뷰 관리 (블라인드 등)

---

## 5. ❤️ 좋아요 기능

* 피드 좋아요 (toggle 방식)
* 좋아요 수 실시간 반영
* 좋아요 누른 사용자 리스트 보기 (`/feed/:feedId/likes`)
* 좋아요한 사용자 프로필로 이동 가능
